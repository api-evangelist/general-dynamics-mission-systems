# General Dynamics Mission SystemsThis is a repo for managing machine readable artifacts for the [General Dynamics Mission Systems APIs](https://gdmissionsystems.com/). The goal is to maintain and evolve an OpenAPI, Postman Collection, and APIs.json index for the [General Dynamics Mission Systems APIs](https://gdmissionsystems.com/).They are a constant work in progress, and will be updated on a regular basis when new information is added, or more of the specifications are supported.Feel free to submit a pull request to evolve any of the artifacts, and leave GitHub issues if you have any questions, comments, or feedback.- [Website](https://gdmissionsystems.com/)- [Developer](https://gdmissionsystems.com/)- [Twitter](https://twitter.com/GDMS)The [APIs.json](https://github.com/api-evangelist/general-dynamics-mission-systems/blob/master/apis.json) index provides links to all the relevant links for the General Dynamics Mission Systems APIs, including the OpenAPI and Postman if they are present.
